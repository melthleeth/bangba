# SUB 3: 블로그 심화 기능 개발

# 개발 일정

#### 1주차

- 아이디어 기획
- 요구사항 명세서 작성

#### 2주차

- 화면 설계
- 디자인 목업
- 기능 명세서 작성, DB설계

#### 3주차

- AWS에 DB 연동
- 웹사이트 뼈대 구축
- 로그인 및 레시피 등 기본기능 구현

#### 4주차

- 핵심기능 개발
- 프론트엔드 (Vue.js)와 백엔드 (Java Spring) 연동
- 테스트 시나리오 및 웹사이트 중간 배포

#### 5주차

- 필수기능 개발 (추천, 마이페이지 등)

#### 6주차

- 버그 픽스
- 테스트 데이터 삽입
- 최종 배포
